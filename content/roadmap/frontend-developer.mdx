# 동작 절차 모르고 써보기

function 키워드 앞에 async를 붙이고 비동기로 처리할 부분에 await를 붙이면 된다.

```javascript
async function main() {
  await delay(1000);
  await delay(2000);
  const result = await Promise.resolve('끝');
  console.log(result);
}

main()
```

# 아하 이렇게 실행되는구나

async/await는 내부적으로 Promise를 사용해서 비동기를 처리한다.
그러면 Promise가 무엇일까?

Promise는 비동기 작업의 최종 완료 또는 실패를 나타내는 Array, Object 같은 객체라고 볼 수 있다.
Promise는 결과를 반환하는게 아니고 상태를 반환한다.
