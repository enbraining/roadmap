# 시작하며

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
async def root():
    return {"message": "Hello World"}
```

다음은 fastAPI의 기초 예제이다.

1. FastAPI를 불러오고,
2. FastAPI 인스턴스를 생성한다.
3. GET 메소드의 "/" 경로로 동작할 것이라고 설정하고,
4. 요청을 하면 Hello World로 반환한다.

# 와!!! 기본적으로 swagger가 있어요
